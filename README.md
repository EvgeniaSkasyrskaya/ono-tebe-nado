### Проект: «Оно тебе надо» – верстка аукционной страницы
Цель работы - сверстать одностраничный сайт для аукциона вещей «Оно тебе надо» с точным соответствием макету, используя современные подходы к веб-разработке.

### Необходимый стек технологий
- HTML5 (семантическая разметка)
- CSS3 (Flexbox, Grid, позиционирование, работа с фоном)
- Методология BEM (для именования классов)
- Pixel Perfect подход (точное соответствие макету)
- Прогрессивный джипег (от общего к частному)

### Этапы выполнения
#### Анализ макета
Изучение структуры макета  
Выявление повторяющихся элементов (оверлей, адрес, меню)  
Определение цветовой палитры и шрифтов  
Анализ готовой заготовки проекта (папки, шрифты, базовые стили)  
#### Формирование базовой структуры документа: 
Подключение CSS-файлов в правильном порядке  
Создание семантической структуры  
#### Глобальная стилизация
Настройка взаимного расположения блоков  
Задание внешних и внутренних отступов  
Установка фоновых изображений для соответствующих блоков  
#### Верстка отдельных секций в соответствии с макетом
#### Финальная проверка
  Тестирование верстки без Live Server  
  Сверка с макетом через Pixel Perfect  
  Проверка семантичности и доступности  
  Оптимизация кода и устранение дублирования  

### Ключевые особенности проекта
- Переиспользуемые компоненты (оверлей, адрес)
- Комбинирование Grid и Flexbox для разных задач
- Семантическая разметка
- Точное соответствие макету (Pixel Perfect)
